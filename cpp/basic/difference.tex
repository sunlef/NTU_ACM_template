一维：区间 $[l, r]$ 每个元素加上 x：
\begin{lstlisting}
d[l] += x;
d[r + 1] -= x;
\end{lstlisting}

二维：$(x1, y1), (x2, y2)$ 每个元素加上x：
\begin{lstlisting}
d[x1][y1] += x;
d[x1][y2 + 1] -= x;
d[x2 + 1][y1] -= x;
d[x2 + 1][y2 + 1] += x;
\end{lstlisting}

三维：$(x1, y1, z1), (x2, y2, z2)$ 每个元素加上x：
\begin{lstlisting}
d[x1][y1][z1] += x;
d[x2 + 1][y1][z1] -= x;
d[x1][y1][z2 + 1] -= x;
d[x2 + 1][y1][z2 + 1] += x;
d[x1][y2 + 1][z1] -= x;
d[x2 + 1][y2 + 1][z1] += x;
d[x1][y2 + 1][z2 + 1] += x;
d[x2 + 1][y2 + 1][z2 + 1] -=x;
\end{lstlisting}
